# Кук не получает зп:
Высокая производительность Кука (efficiency_factor большой)

          Кук успевает готовить много наггетсов
          Тарелки постоянно пополняются

Высокая прожорливость толстяков (gluttony большой)

          Каждый толстяк быстро наедает свои 10000 наггетсов

          Они не успевают умереть от голода

Баланс в пользу переедания

          Кук производит > чем толстяки успевают съесть за один подход

          Но толстяки едят достаточно много, чтобы быстро достичь лимита


# Кука увольняют:
Низкая производительность Кука (efficiency_factor маленький)

            Кук не успевает пополнять тарелки

            Производство < потребления

Высокая прожорливость толстяков (gluttony большой)

            Толстяки быстро опустошают тарелки

            Спрос > предложения

Дисбаланс в пользу голода

            Толстяки едят быстрее, чем Кук готовит

            Одна из тарелок опустошается ДО того, как все толстяки наедают 10000


# Кук уволился сам:
Сбалансированные коэффициенты

            Производство ≈ потребление

            Кук успевает пополнять тарелки

            Толстяки едят умеренно

Стабильная работа системы

            Ни одна тарелка не опустошается

            Ни один толстяк не "лопается"

            Система работает стабильно все 5 дней

Истечение времени

            Проходит ровно 5 дней (5 секунд)

            Кук уходит по собственному желанию
